一朵野花
陈梦家
一朵野花在荒原里开了又落了，
不想到这小生命，向着太阳发笑，
上帝给他的聪敏他自己知道，
他的欢喜，他的诗，在风前轻摇。

一朵野花在荒原里开了又落了，
他看见青天，看不见自己的渺小，
听惯风的温柔，听惯风的怒号，
就连他自己的梦也容易忘掉。
